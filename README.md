# Blagweb Agency

A modern, premium software agency website built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, **MongoDB**, and **Mongoose**.

Blagweb Agency provides digital solutions including web development, mobile applications, UI/UX design, SEO, cloud solutions, CRM systems, ERP software, and digital marketing services.

* Next.js 15 App Router

## 🛠 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/blagweb-agency.git
```

Go to project directory:

```bash
cd blagweb-agency
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
